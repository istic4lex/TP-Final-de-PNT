TP Final PP2 - TV Series

- Integrante: Alex Arone


Tema de la App:
TV Series es una aplicación Android Studio que permite a los usuarios buscar y explorar series de televisión utilizando la API pública de TVMaze.

API Utilizada:
- Nombre: TVMaze API
- URL: https://www.tvmaze.com/api

Instrucciones para correr la app:
1. Clonar el repositorio
2. Abrir el proyecto en Android Studio
3. Sincronizar Gradle
4. Ejecutar en el emulador o dispositivo físico
